# 网站项目大纲

## 文件结构

### 核心HTML页面
- **index.html** - 主页
  - Hero区域：个人介绍与专业背景
  - 摄影作品精选预览（轮播）
  - 最新博客文章摘要
  - 个人成就展示
  
- **gallery.html** - 摄影画廊
  - 作品分类筛选器（微距、生态、实验等）
  - 瀑布流图片展示
  - 灯箱全屏预览功能
  - 搜索功能
  
- **blog.html** - 个人博客
  - 文章列表（卡片式布局）
  - 分类侧边栏
  - 文章详情页
  - 标签系统
  
- **resources.html** - 生物资源链接
  - 链接分类管理
  - 收藏功能
  - 搜索筛选
  - 外部网站跳转

### 资源文件
- **resources/** - 图片和媒体资源文件夹
  - hero-biologist.png - 主页hero图像
  - gallery/ - 摄影作品图片
  - blog/ - 博客文章配图
  - icons/ - 网站图标

### JavaScript文件
- **main.js** - 主要交互逻辑
  - 页面导航控制
  - 图片画廊功能
  - 博客系统交互
  - 动画效果控制

## 页面功能详述

### 主页 (index.html)
1. **导航栏**
   - 响应式导航菜单
   - 当前页面高亮
   - 移动端汉堡菜单

2. **Hero区域**
   - 个人专业照片背景
   - 动画标题文字效果
   - 简介文字和CTA按钮

3. **作品预览**
   - Splide.js轮播展示
   - 悬停效果和链接跳转
   - 自动播放控制

4. **博客摘要**
   - 最新3篇文章预览
   - 阅读更多链接
   - 发布日期显示

### 摄影画廊 (gallery.html)
1. **筛选系统**
   - 分类标签（全部、微距、生态、实验、野外）
   - 实时筛选动画
   - 搜索结果计数

2. **图片网格**
   - 响应式瀑布流布局
   - 图片懒加载
   - 悬停信息显示

3. **灯箱功能**
   - 全屏图片预览
   - 左右导航箭头
   - 图片信息展示
   - 键盘导航支持

### 博客系统 (blog.html)
1. **文章管理**
   - 分类筛选
   - 标签云显示
   - 搜索功能
   - 分页加载

2. **阅读体验**
   - 清晰的文章布局
   - 代码高亮（如需要）
   - 相关文章推荐
   - 社交分享按钮

### 资源链接 (resources.html)
1. **链接管理**
   - 分类展示（期刊、机构、工具等）
   - 收藏功能（本地存储）
   - 搜索和筛选
   - 访问统计

## 技术实现

### 前端框架
- 纯HTML5 + CSS3 + JavaScript
- Tailwind CSS用于样式
- 响应式设计原则

### 核心库集成
- Anime.js - 动画效果
- ECharts.js - 数据可视化
- Splide.js - 轮播组件
- p5.js - 创意背景
- Pixi.js - 视觉特效
- Matter.js - 物理交互
- Shader-park - 着色器效果

### 性能优化
- 图片懒加载
- CSS和JS压缩
- 缓存策略
- 移动端优化
