# 网站交互设计

## 核心交互功能

### 1. 摄影作品展示画廊
- **图片网格展示**: 使用瀑布流布局展示摄影作品，支持分类筛选（微距摄影、生态摄影、实验摄影等）
- **灯箱效果**: 点击图片可全屏预览，支持左右滑动浏览，显示拍摄参数和描述
- **分类筛选器**: 顶部标签栏，支持多选筛选，实时更新展示内容
- **搜索功能**: 根据关键词搜索图片标题和描述

### 2. 个人博客系统
- **文章列表**: 卡片式布局显示文章预览，包含封面图、标题、摘要、发布时间
- **分类导航**: 侧边栏显示博客分类（科研随笔、摄影技巧、野外考察、生活感悟等）
- **文章阅读**: 点击文章卡片进入详细阅读页面，支持返回列表
- **标签系统**: 文章标签显示，支持点击标签筛选相关文章

### 3. 生物资源链接库
- **链接分类**: 按类型分组（学术期刊、研究机构、野外指南、在线工具等）
- **收藏功能**: 用户可以点击收藏喜欢的链接，本地存储
- **搜索筛选**: 支持按名称、描述搜索链接
- **外部跳转**: 点击链接在新窗口打开外部网站

### 4. 个人介绍页面
- **时间线展示**: 使用交互式时间线展示学术和摄影经历
- **技能展示**: 使用进度条和图标展示专业技能
- **联系方式**: 集成联系表单，支持发送邮件

## 用户体验设计
- 响应式设计，适配移动端和桌面端
- 流畅的页面切换动画
- 悬停效果和微交互
- 无限滚动加载内容
- 图片懒加载优化性能
